About Book Explorer

Book Explorer is a responsive web application designed to help users discover books from around the world using the Open Library API.

Users can search books, load more books, save favorites using localStorage, and browse books across multiple responsiv pages.

Technologies Used

  • ✅ HTML5
  • ✅ Tailwind CSS
  • ✅ JavaScript ES6 Modules
  • ✅ Open Library API
  • ✅ localStorage
